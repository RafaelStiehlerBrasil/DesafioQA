Desafio QA

PRÉ-REQUISITOS
* Python 9
* Pytest
* Pytest-html-reporter 0.2.6

INSTALAÇÃO
* Instalar o python https://www.python.org/downloads/
* Abrir terminal e rodar o comando pip install requests pytest
* Abrir terminal e rodar o comando pip install pytest-html-reporter
* Instalar Visual Studio Code https://code.visualstudio.com/download
* Abrir Visual Studio Code File > Open Folder e selecionar o projeto desafio QA

EXECUTANDO OS TESTES
* Abrir terminal no Visual Studio Code e rodar o comando pytest
* Para gerar o relatório de falhas rodar o comando pytest --html-report=./report/report.html

VERSÃO 
1.0.0
